## ADDED Requirements

### Requirement: Загрузка фото для распознавания

FaceRecognitionContent SHALL предоставлять интерфейс загрузки фотографии для распознавания лиц. Распознавание выполняется через ServiceLocator.faceRecognitionRepository.recognizeFace(imageBytes).

#### Scenario: Успешная загрузка и распознавание

- **WHEN** пользователь выбирает файл фотографии и нажимает «Распознать»
- **THEN** фото отправляется на сервер, возвращаются результаты распознавания

#### Scenario: Ошибка распознавания

- **WHEN** распознавание завершается ошибкой
- **THEN** отображается сообщение об ошибке на русском языке (через ErrorMapper)

#### Scenario: Файл не выбран

- **WHEN** пользователь нажимает «Распознать» без выбора файла
- **THEN** кнопка неактивна или отображается предупреждение «Выберите файл»

### Requirement: Отображение результатов распознавания

FaceRecognitionContent SHALL отображать результаты распознавания: количество обнаруженных лиц (faces_detected) и список совпадений для каждого лица с username, similarity и bounding box.

#### Scenario: Лица обнаружены

- **WHEN** распознавание успешно и лица найдены
- **THEN** отображается список результатов: для каждого лица — bounding box, совпадения с именами и процентом сходства

#### Scenario: Лица не обнаружены

- **WHEN** распознавание успешно, но лица не найдены
- **THEN** отображается сообщение «Лица не обнаружены»

### Requirement: Предпросмотр изображения

FaceRecognitionContent SHALL отображать предпросмотр выбранного изображения перед распознаванием.

#### Scenario: Выбор файла

- **WHEN** пользователь выбирает файл изображения
- **THEN** предпросмотр изображения отображается в UI

### Requirement: Индикатор загрузки при распознавании

При выполнении распознавания FaceRecognitionContent SHALL отображать индикатор загрузки.

#### Scenario: Распознавание в процессе

- **WHEN** запрос на распознавание отправлен на сервер
- **THEN** отображается индикатор загрузки до получения результата
