## Почему

Экран распознавания лиц («Лица») поддерживает только выбор фото из файла и отображает результаты в виде текста — координаты bounding box показаны числами, что не даёт визуального представления о том, какое лицо какому совпадению соответствует. Инфраструктура захвата с камеры (WebcamService, PhotoCaptureDialog) уже существует для регистрации сотрудников, но не используется для распознавания лиц. Настройка `matchThreshold`, управляющая чувствительностью распознавания, не представлена в UI настроек — оператор не может настроить порог без доступа к серверу.

## Что изменяется

- Добавление кнопки захвата фото с веб-камеры на экране распознавания лиц — повторное использование существующих `PhotoCaptureDialog` и `WebcamService`, чтобы пользователь мог сфотографировать лицо напрямую вместо выбора файла
- Отрисовка bounding box поверх изображения предпросмотра — прямоугольники с подписями (имя лучшего совпадения, процент сходства), цветовые кодировки по качеству совпадения (зелёный >80%, жёлтый 60–80%, красный <60%)
- Добавление настроек распознавания лиц в диалог настроек — слайдер `matchThreshold`, поля `cameraResolution` и `cameraFps`, сохранение через `SettingsRepository`

## Возможности

### Новые возможности
- `face-recognition-settings`: Управление параметрами распознавания лиц (слайдер matchThreshold, поля cameraResolution и cameraFps) в UI настроек с сохранением на сервер через SettingsRepository

### Изменяемые возможности
- `face-recognition-ui`: Добавление режима захвата с камеры и визуализации bounding box на экране распознавания лиц

## Влияние

- `FaceRecognitionScreen.kt` — добавление кнопки захвата с камеры, интеграция PhotoCaptureDialog, добавление overlay для bounding box
- `SettingsOverlay.kt` — добавление секции настроек распознавания лиц
- `SettingsState.kt` — добавление наблюдаемых полей состояния для настроек распознавания, загрузка и сохранение
- `FaceBoundingBoxOverlay.kt` — новый composable для отрисовки bounding box поверх изображения
- Существующие доменные модели (`FaceRecognitionResult`, `BoundingBox`) — не требуют изменений
- Существующие сервисы (`WebcamService`, `PhotoCaptureDialog`) — используются без изменений
