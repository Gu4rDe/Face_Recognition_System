## MODIFIED Requirements

### Requirement: HTTP 403 отображает detail сообщение
При HTTP 403 ошибке клиент должен отображать detail сообщение от сервера вместо общего текста.

#### Scenario: Сервер возвращает detail
- **WHEN** сервер возвращает HTTP 403 с JSON `{"detail": "заблокирован"}`
- **THEN** клиент отображает "заблокирован"

#### Scenario: Сервер не возвращает detail
- **WHEN** сервер возвращает HTTP 403 без поля detail
- **THEN** клиент отображает "Доступ запрещён"
