## ADDED Requirements

### Requirement: Настройка порога распознавания лиц

SettingsOverlay SHALL предоставлять управление порогом совпадения (matchThreshold) через Slider в диапазоне от 0.0 до 1.0 с шагом 0.01. Текущее значение SHALL отображаться текстом рядом со слайдером в формате процентов (напр. «75%»). Порог используется FaceBoundingBoxOverlay для цветовой кодировки bounding box.

#### Scenario: Загрузка текущего порога

- **WHEN** диалог настроек открывается
- **THEN** слайдер matchThreshold отображает текущее значение, загруженное с сервера через SettingsRepository.getSettings()

#### Scenario: Изменение порога

- **WHEN** пользователь перемещает слайдер matchThreshold
- **THEN** текстовое отображение обновляется немедленно, но сохранение на сервер происходит только при закрытии диалога

#### Scenario: Сохранение порога на сервер

- **WHEN** пользователь закрывает диалог настроек
- **THEN** значение matchThreshold отправляется на сервер через SettingsRepository.updateSettings() в составе AppSettingsUpdate

### Requirement: Настройка параметров камеры

SettingsOverlay SHALL предоставлять поля для настройки разрешения камеры (cameraResolution, строка формата «WxH») и частоты кадров камеры (cameraFps, целое число).

#### Scenario: Загрузка параметров камеры

- **WHEN** диалог настроек открывается
- **THEN** поля cameraResolution и cameraFps отображают текущие значения, загруженные с сервера

#### Scenario: Изменение разрешения камеры

- **WHEN** пользователь вводит новое значение в поле cameraResolution
- **THEN** значение обновляется в локальном состоянии, сохранение происходит при закрытии диалога

#### Scenario: Изменение частоты кадров

- **WHEN** пользователь вводит новое значение в поле cameraFps
- **THEN** значение обновляется в локальном состоянии, сохранение происходит при закрытии диалога

### Requirement: Группировка настроек распознавания лиц

Настройки распознавания лиц (matchThreshold, cameraResolution, cameraFps) SHALL быть сгруппированы в отдельную секцию с заголовком «Распознавание лиц» в диалоге настроек.

#### Scenario: Отображение секции настроек распознавания

- **WHEN** диалог настроек открыт
- **THEN** секция «Распознавание лиц» отображается после существующих настроек (тема, API URL) и содержит слайдер matchThreshold, поле cameraResolution и поле cameraFps

### Requirement: Обратная связь при ошибках настроек

SettingsOverlay SHALL отображать сообщения об ошибках загрузки и сохранения настроек распознавания лиц.

#### Scenario: Ошибка загрузки настроек

- **WHEN** загрузка настроек с сервера завершается ошибкой
- **THEN** в секции распознавания лиц отображается сообщение об ошибке

#### Scenario: Ошибка сохранения настроек

- **WHEN** сохранение настроек на сервер завершается ошибкой
- **THEN** в секции распознавания лиц отображается сообщение об ошибке
