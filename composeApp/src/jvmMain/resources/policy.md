# 📄 Политика обработки персональных данных

## (система идентификации по изображению лица для учёта посещаемости)

---

## 1. Общие положения

1.1. Настоящая Политика регулирует обработку персональных данных при использовании системы идентификации по изображению лица (далее — «Система») для учёта посещаемости.

1.2. Система применяется для:

* регистрации факта присутствия обучающихся;
* контроля доступа в помещения;
* ведения учёта посещаемости.

1.3. Оператором персональных данных является организация, внедрившая Систему (образовательное учреждение или иная организация).

---

## 2. Состав обрабатываемых данных

В рамках работы Системы обрабатываются:

* фамилия, имя, отчество
* изображение лица
* биометрические персональные данные (данные, характеризующие физиологические особенности лица)
* сведения о посещаемости (дата и время фиксации)

---

## 3. Правовые основания обработки

3.1. Обработка персональных данных осуществляется в соответствии с Федеральным законом РФ №152-ФЗ «О персональных данных».

3.2. Основанием для обработки является:
👉 **письменное согласие субъекта персональных данных**

3.3. Обработка биометрических персональных данных без согласия субъекта не допускается, за исключением случаев, предусмотренных законодательством РФ.

---

## 4. Цели обработки данных

Персональные данные обрабатываются исключительно для:

* учёта посещаемости
* контроля доступа
* обеспечения внутреннего порядка

❗ Использование данных для иных целей не допускается.

---

## 5. Порядок получения согласия

5.1. До начала обработки персональных данных субъект должен предоставить письменное согласие.

5.2. Согласие должно содержать:

* перечень обрабатываемых данных
* цели обработки
* сроки хранения
* порядок отзыва

5.3. При отказе от предоставления согласия субъекту должен быть обеспечен альтернативный способ учёта посещаемости.

---

## 6. Условия обработки и хранения данных

6.1. Обработка осуществляется с соблюдением принципов:

* законности
* минимизации данных
* ограничения сроков хранения

6.2. Данные хранятся:

* на защищённых носителях
* с ограниченным доступом

6.3. Срок хранения:

* биометрические данные — не дольше срока достижения целей обработки
* сведения о посещаемости — в соответствии с внутренними регламентами

---

## 7. Обеспечение безопасности данных

Оператор принимает необходимые меры для защиты данных, включая:

* ограничение доступа к данным
* использование средств защиты информации
* предотвращение несанкционированного доступа

---

## 8. Передача данных третьим лицам

8.1. Передача персональных данных допускается только:

* с согласия субъекта
* либо в случаях, предусмотренных законодательством РФ

8.2. Передача данных третьим лицам без законных оснований запрещена.

---

## 9. Права субъекта персональных данных

Субъект персональных данных имеет право:

* получать информацию об обработке своих данных
* требовать уточнения, блокирования или удаления данных
* отозвать согласие на обработку

---

## 10. Обязанности оператора

Оператор обязан:

* соблюдать требования законодательства РФ
* обеспечивать конфиденциальность данных
* принимать меры по защите данных
* прекращать обработку по требованию субъекта

---

## 11. Ответственность

Лица, допустившие нарушение требований законодательства о персональных данных, несут ответственность в соответствии с законодательством Российской Федерации.

---

## 12. Заключительные положения

12.1. Использование Системы допускается только при соблюдении настоящей Политики.
12.2. Политика подлежит обновлению при изменении законодательства РФ или условий обработки данных.

---
